# Panduan Izin Kamera dan Lokasi

## Masalah Umum
Ketika aplikasi absensi dibuka di HP/mobile browser, browser akan meminta izin untuk mengakses:
1. **Lokasi GPS** - untuk memverifikasi posisi karyawan
2. **Kamera** - untuk mengambil foto selfie absensi

## Cara Mengizinkan Akses

### Android - Chrome
1. Buka aplikasi di browser Chrome
2. Ketika muncul pop-up "Allow [nama-website] to access your location/camera?", klik **Allow**
3. Jika tidak muncul pop-up atau sudah ditolak sebelumnya:
   - Klik ikon **gembok** atau **info (i)** di sebelah kiri address bar
   - Pilih **Site settings** atau **Permissions**
   - Cari **Location** dan **Camera**
   - Ubah dari "Blocked" menjadi **"Allow"**
   - Refresh halaman

### Android - Firefox
1. Buka aplikasi di browser Firefox
2. Ketika muncul pop-up izin, klik **Allow**
3. Jika sudah ditolak:
   - Klik ikon **gembok** di address bar
   - Pilih **Edit Site Settings**
   - Ubah **Location** dan **Camera** menjadi **"Allow"**
   - Refresh halaman

### iOS - Safari
1. Buka aplikasi di Safari
2. Ketika muncul pop-up izin, klik **Allow**
3. Jika sudah ditolak:
   - Buka **Settings** > **Safari** > **Camera** dan **Location**
   - Pastikan diset ke **"Ask"** atau **"Allow"**
   - Atau buka **Settings** > **Privacy & Security**
   - Pilih **Location Services** dan **Camera**
   - Cari Safari dan pastikan diizinkan
   - Refresh halaman di Safari

### iOS - Chrome
1. Buka aplikasi di Chrome
2. Ketika muncul pop-up izin, klik **Allow**
3. Jika sudah ditolak:
   - Buka **Settings** > **Chrome** > **Camera** dan **Location**
   - Pastikan diset ke **"Allow"**
   - Refresh halaman

## Troubleshooting

### Lokasi Tidak Terdeteksi
- Pastikan **GPS/Location Services** aktif di pengaturan HP
- Pastikan koneksi internet stabil
- Coba refresh halaman
- Pastikan browser memiliki izin lokasi

### Kamera Tidak Berfungsi
- Pastikan tidak ada aplikasi lain yang menggunakan kamera
- Tutup aplikasi kamera lain yang sedang berjalan
- Restart browser
- Pastikan browser memiliki izin kamera

### Error "Permission Denied"
- Hapus cache dan cookies browser
- Buka halaman dalam mode **Incognito/Private** untuk test
- Jika masih error, uninstall dan install ulang browser
- Pastikan browser sudah versi terbaru

## Keamanan
- Aplikasi hanya mengakses kamera dan lokasi saat proses absensi
- Data lokasi dan foto hanya dikirim ke server perusahaan
- Tidak ada data yang disimpan di browser atau perangkat
